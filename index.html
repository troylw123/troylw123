My Projects

Nerdle

 C#

Nerdle was my first original project as a developer. I worked with a partner at Eleven Fifty Academy to create a simple number-guessing terminal game based on the same concept as the popular word-guessing game Wordle. A random 3 digit number is generated and after each guess the game tells you higher, lower or correct for each digit you guessed. There is certainly room for improvement, but the game is actually a lot of fun. The code is written entirely in C#.

CodeBusters

 C#

CodeBusters is a project I completed as part of a four person team at Eleven Fifty Academy. It's a WebAPI that contains 6 distinct tables, each with at least 4 working HTTP endpoints. The coding was done in C# and utilizes N-Tier Architecture and Microsoft Entity Framework. Running the project requires an SQL server, Azure Data Studio, and either Postman or Swagger. Those are all technologies I learned to use as part of this project. As a stretch goal, I created a console application for CodeBusters, which includes the CRUD operations for the User table.
